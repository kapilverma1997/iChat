# Notification Sounds

Place your notification sound files in this directory.

## Recommended Sound File

- **notification.mp3** - Main notification sound (recommended format: MP3, duration: 0.3-0.5 seconds)

## Alternative Formats

You can use any of these formats:
- MP3 (`.mp3`)
- WAV (`.wav`)
- OGG (`.ogg`)

## Getting Sound Files

You can download free notification sounds from:
- [Freesound.org](https://freesound.org/)
- [Zapsplat](https://www.zapsplat.com/)
- [Notification Sounds](https://notificationsounds.com/)

## Fallback

If no sound file is found, the system will use a Web Audio API beep as a fallback.
